選擇標準

1. 第一眼信任感日料店老板能否在 3 秒內判斷 "這是專業供應商"
2. 信息密度B2B 買家需要的規格、MOQ、溯源是否觸手可及
3. 品牌氣質和日料文化是否契合，是否區別於普通食材站
4. Wix 遷移難度當前方向能否在 Wix Editor / Studio 內實現
YOUR CHOICE · 最終版

變體 D · A hero + C 產品卡

混搭 A 的震撼 hero + C 的 B2B 效率，補齊 4 個產品 / 關於 / 溯源子頁面。導航可點、牧場圖已修、電話字體正常、資質條重排。

A — Poseidon Hero（參考）
全幅大圖英雄
Cinematic · Editorial-Luxury
結構：全屏 hero 大圖 → 品牌介紹 + 4 統計 → 3 產品網格 → 溯源區塊 → 聯絡。
  • 氣質：Michelin Guide / psdgp.com 式
  • 強在：第一眼震撼，高端餐飲文化契合度最高
  • 弱在：信息密度較低，買家要滾一屏才看到產品

B — Editorial Spread（參考）
編輯刊物式
Kinfolk · Magazine
結構：Hero 左圖右文 + Issue 刊號 → 01 Wagyu / 02 Uni / 03 Seafood 雜誌專題頁 → 引文 + 規格表 → 聯絡。
  • 氣質：Kinfolk 雜誌 + 米芝蓮指南內頁
  • 強在：長文可承載品牌故事與規格，最適合講"匠人"敘事
  • 弱在：頁面長，買家想直接看產品會不耐煩

C — Grid Catalogue（參考）
3 欄產品網格
B2B · Efficient
結構：頂部"本週到貨"動態條 → 首屏直接 3 產品卡（含 MOQ / Grade / Lead）→ 資質條 → 合作餐廳 → 3 服務 → 黑色 CTA 條。
  • 氣質：高端 B2B / The Outnet 式效率優先
  • 強在：首屏信息最密，買家下單轉化最快
  • 弱在：相對"冷"，品牌情感弱於 A/B
